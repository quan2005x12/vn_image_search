# vietnamese_food_images_searcher

![Python 3.12.6](https://img.shields.io/badge/Python-3.12.6-blue?logo=python&logoColor=white)

## Instructions

1. Download [dataset](https://kaggle.com/datasets/7346112bd4ac65c71e403b31d9d289bbb9b533fff7746261054562ffb0435e45)
2. Run `vector_v3.py` to extract and save the vectors (có rồi thì bỏ qua)
3. Run `app_v3.py` to start the API
4. Open http://localhost:5000 with your browser
